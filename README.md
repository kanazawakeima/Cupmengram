# README
# 開発言語
  - Ruby 3.0.1
  - rails 6.0.3
# 就職Termの技術
  - devise
  - お気に入り機能
# カリキュラム外の技術
  - ransack
# 実行手順

```
   $ git clone git@github.com:kanazawakeima/Cupmengram.git
   $ cd Cupmengram
   $ bundle install
   $ rails db:create && rails db:migrate
   $ rails s
```
